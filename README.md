# Horizon Beleggingen Screener
Horizon Beleggingen Screener voor aandelen, ETF's en meer. 

## Get Started
Applicatie bekijken:

```
npm install
npm run start
```

Meehelpen aan de applicatie: 

```
npm install
npm run dev
```
