# Horizon Beleggingen Screener
Horizon Beleggingen Screener voor aandelen, ETF's en meer. 

## Applicatie gebruiken
Applicatie bekijken:

```
git clone https://github.com/rohitjethoe/horizon-beleggen.git
npm install
npm run start
```

Meehelpen aan de applicatie: 

```

npm install
npm run dev
```
